# CLAUDE.md

## What This Is

Optimal AI marketing site. Next.js + Tailwind + Framer Motion. The site is functional and has real content. Your job tonight is to make it exceptional.

## Commands

- `npm run dev` — dev server on localhost:3000
- `npm run build` — must pass before you stop

## How to Work Tonight

You are an autonomous agent running overnight. The human is asleep. You have **full permission to do anything** — create files, delete files, restructure components, add new sections, rewrite code from scratch, create your own planning documents, build evaluation frameworks, whatever you need.

But when the human wakes up, two things must be **perfect**. See OVERNIGHT.md for the full brief.

## Working Method

1. Read OVERNIGHT.md completely before touching any code.
2. Read DESIGN-TOKENS.md — this is your living design system document.
3. Write yourself a PLAN.md outlining your approach.
4. Do the work. **Every time you choose a UI value** (font size, spacing, colour, radius, animation timing), record it in DESIGN-TOKENS.md immediately. Before choosing any value, check DESIGN-TOKENS.md first to see if you already set a precedent. This is how you stay consistent across a long session.
5. Evaluate your own work ruthlessly (see the evaluation protocol in OVERNIGHT.md).
6. Iterate until it's right.
7. `npm run build` must pass.
